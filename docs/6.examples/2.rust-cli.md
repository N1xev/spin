## spin.toml

```toml
name = "rust-cli"
version = "0.1.0"
description = "Minimal Rust CLI starter"
type = "cli"
language = "rust"

[params]
project_name = { type = "text", prompt = "Project name", default = "mycli" }
edition = { type = "select", prompt = "Rust edition", options = ["2021", "2024"], default = "2021" }

[[post]]
run = "cargo check"
```

## _base/Cargo.toml.tmpl

```toml
[package]
name = "{{ .project_name | snake_case }}"
version = "0.1.0"
edition = "{{ .edition }}"

[dependencies]
clap = { version = "4", features = ["derive"] }
```

## _base/src/main.rs.tmpl

```rust
use clap::Parser;

#[derive(Parser)]
#[command(name = "{{ .project_name | kebab }}")]
struct Cli {
    name: Option<String>,
}

fn main() {
    let cli = Cli::parse();
    match cli.name {
        Some(name) => println!("Hello, {}!", name),
        None => println!("Hello from {{ .project_name }}!"),
    }
}
```

## Usage

```sh
spin new myapp --template ./rust-cli-template
```
