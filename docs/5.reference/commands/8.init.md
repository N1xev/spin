## Usage

```sh
spin init <name>
```

| Flag | Description |
| --- | --- |
| `--dir` | Parent directory to create the template in (default: current dir). |

## Examples

```sh
spin init my-cli-template
spin init my-template --dir ./templates
```

## Output

Creates:

```
<name>/
  spin.toml
  _base/
    file.txt.tmpl
  README.md
```

Edit `spin.toml` and `_base/` to build your template. Test it with `spin new <project> --template ./<name> --dry-run`.
