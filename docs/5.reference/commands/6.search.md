## Usage

```sh
spin search <query>
```

| Flag | Description |
| --- | --- |
| `--limit` | Max results to show (default: 20). |
| `--json` | Machine-readable output. |

## Examples

```sh
spin search "go cli"
spin search rust --limit 5
spin search tauri --json
```

## Results

Results are sorted by relevance:

1. Exact `<alias>/<id>` match
2. `<alias>/<id>` substring match
3. Name match
4. Description match
5. Tag match

## Add a result

```sh
spin add official/go-cli
spin new myapp --template official/go-cli
```
