## Usage

```sh
spin add <spec>
spin add --list
```

| Flag | Description |
| --- | --- |
| `--list` | Show pinned templates and exit. |

## Spec

- Git URL: `https://github.com/example/go-cli-template.git`
- GitHub shorthand: `example/go-cli-template`
- Local path: `~/templates/go-cli`
- Registry shorthand: `official/go-cli`

## Examples

```sh
spin add https://github.com/example/go-cli-template.git
spin add ~/templates/go-cli
spin add official/go-cli
spin add --list
```

## Storage

Pins are recorded in `~/.config/spin/pinned.json`. The cached clone lives in `~/.config/spin/templates/<name>/`.
