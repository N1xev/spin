## Usage

```sh
spin update [name]
```

## Examples

```sh
spin update go-cli-template
spin update
```

Omit `name` to refresh every pinned template.

## Rollback

`spin update` moves the old cache aside before refreshing. If the refresh or pin write fails, it attempts to restore the old cache.
