1. Add a new method to `internal/registry/client.go`:
   - `Add(spec string) (*Pinned, error)`. Logic:
     a) If `spec` is a local path (starts with `/`, `.`, or `~`): validate the dir exists, compute the cache target as `filepath.Join(c.CacheDir, "templates", basename(spec))`, and either symlink (preferred) or recursively copy. Use `os.Symlink` first; if it fails (e.g. on Windows without privs), fall back to `cp -R` via `os/exec` (or write a small `copyDir` helper using `os.ReadDir` + `os.WriteFile`). Write the Pinned record with `Source: spec, Version: "local", LocalPath: cacheTarget`.
     b) If `spec` is a git URL (starts with `http://`, `https://`, `git@`, `git://`, `ssh://`): shallow-clone to `c.CacheDir/templates/<repo-name>` using `exec.Command("git", "clone", "--depth=1", spec, dest)` with `cmd.Env = append(os.Environ(), "GIT_TERMINAL_PROMPT=0")`. Write the Pinned record with `Source: spec, Version: "git", LocalPath: dest`.
     c) Otherwise (assumed `user/repo` shorthand for the public registry, which is not deployed yet): return a clear error `"registry shorthand %q is not yet supported; use a full git URL or a local path"`. Do NOT try to look it up against the unreachable registry.
   - Update the `Pinned` struct in `types.go` to add a `LocalPath string` field (JSON tag `local_path`). The pin file is append-only; existing pins without `LocalPath` still work.
   - Update `Pin(p Pinned)` to require `LocalPath` is set (or set it from a default if empty: `filepath.Join(c.CacheDir, "templates", p.Name)`).

2. Rewrite `cmd/add.go` `runAdd`:
   - Build the client: `c := registry.New()`.
   - Call `pinned, err := c.Add(args[0])`.
   - If success, `c.Pin(*pinned)` to persist. (Note: `Add` may clone; `Pin` writes the JSON. The order matters -- clone first, then write JSON only on success.)
   - Print `✓ added <name> (cloned to <local_path>)` or `✓ added <name> (local at <local_path>)`.
   - If `args[0]` is empty or `--list` flag is set: print the pinned list (delegate to `execList` logic in `cmd/list.go`). Add the `--list` flag to `addCmd`.

3. Update `cmd/list.go` `execList`:
   - For each pinned entry, also print the `LocalPath` column.
   - Add a "Refresh" hint: `Run ` + "`spin new <name> --refresh`" + ` to re-clone.` (The --refresh flag is a future enhancement; for v2.0 the hint is informational only.)

Use only `os`, `os/exec`, `path/filepath`, `time`, plus the existing imports. No new dependencies.
