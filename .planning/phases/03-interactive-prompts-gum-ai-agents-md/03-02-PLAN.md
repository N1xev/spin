## Phase Goal (as user story)

**As a** `spin` user without `gum` installed who runs `spin new` interactively, **I want** `spin` to ask me for the missing fields via in-process TUI forms, **so that** I get the same guided experience without needing an extra binary.

```go
// AllLibs returns the unified library set: every entry in p.Libs plus
// every bool set in the boolFlagOverlayMap (Cobra, Fang, Viper, Huh,
// Glamour, Glow, Wish, Log, Harmonica), deduplicated and sorted
// alphabetically. Use this anywhere a "complete library list" is needed:
// the AGENTS.md template, the prompt default-selection, and the overlay
// walker's order. Returns a fresh slice; safe to mutate.
func (p *Project) AllLibs() []string {
    seen := map[string]bool{}
    var out []string
    for _, lib := range p.Libs {
        if !seen[lib] { seen[lib] = true; out = append(out, lib) }
    }
    for lib, active := range p.boolFlagOverlayMap() {
        if active && !seen[lib] { seen[lib] = true; out = append(out, lib) }
    }
    sort.Strings(out)
    return out
}
```

```go
func askXxx(p *scaffold.Project) error {
    var v string
    err := huh.NewInput[string]().
        Title("...").
        Description("...").
        Value(&v).
        Run()
```
