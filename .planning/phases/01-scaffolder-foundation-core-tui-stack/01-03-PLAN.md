       - Initialize `out := map[string][]byte{}`
       - For each `layer` in `p.overlayOrder()`:
         - Walk `FS` under `templates/<layer>` (use `fs.WalkDir(FS, "templates/<layer>", func(path string, d fs.DirEntry, err error) { ... })`)
         - For each file ending in `.tmpl`: compute `relPath := strings.TrimPrefix(path, "templates/<layer>/")` then `outRelPath := strings.TrimSuffix(relPath, ".tmpl")`. If `outRelPath` is `.tmpl`-stripped, the file is overwritten in `out` (last-write-wins).
         - Read file content via `fs.ReadFile(FS, path)`. If error, return it.

         - Store: `out[outRelPath] = buf.Bytes()`
       - Return out.
